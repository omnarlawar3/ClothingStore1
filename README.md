# Online Clothing Store

This is a Java Web Application for an Online Clothing Store.

## Technologies Used
- Java
- JSP
- Servlet
- HTML
- CSS
- Oracle Database
- Apache Tomcat

## Features
- User Login and Registration
- Product Listing
- Add to Cart
- Order System
- Admin Panel

## Tools
- Eclipse IDE
- Apache Tomcat Server
- Oracle Database
